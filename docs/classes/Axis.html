getTicksCoords

  • getTicksCoords(opt?: { clamp?: boolean; tickModel?: Model<any> }): TickCoord[]
    • Defined in node_modules/echarts/types/dist/shared.d.ts:2841

    Different from zrUtil.map(axis.getTicks(), axis.dataToCoord, axis), axis.getTicksCoords considers onBand, which is used by boundaryGap:true of category axis and splitLine and splitArea.

    Parameters

    • Optional opt: { clamp?: boolean; tickModel?: Model<any> }
      • Optional clamp?: boolean

        If true, the first and the last tick must be at the axis end points. Otherwise, clip ticks that outside the axis extent.

      • Optional tickModel?: Model<any>

        default: axis.model.getModel('axisTick')

    Returns TickCoord[]
